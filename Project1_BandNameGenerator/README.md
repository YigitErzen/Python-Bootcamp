# Project 1: Band Name Generator

## Description

This is a simple Python application that generates a creative band name by asking the user two simple questions. It combines the name of the city where the user grew up with their pet's name to suggest a fun band name.

## Features

- Prompts user for the name of the city they grew up in
- Prompts user for their pet's name
- Combines these two pieces of information to create a unique band name

## How to Run

1. Make sure Python is installed on your computer (Python 3.x recommended)
2. Open terminal or command prompt
3. Navigate to the project directory:
   ```bash
   cd Project1_BandNameGenerator
   ```
4. Run the program:
   ```bash
   python main.py
   ```

## Usage Example

```
Welcome to the Band Name Generator.
What's the name of the city you grew up in?
Istanbul
What's your pet's name?
Mavi
Your band name could be Istanbul Mavi
```
